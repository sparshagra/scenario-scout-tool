# WhatIF Chatbot Frontend

This directory contains the frontend source code for the "WhatIF Chatbot & AI-Powered Document Analysis" application.

It is responsible for rendering the user interface, managing user interactions, and communicating with the backend API.

---

## Prerequisites

Before you begin, ensure you have the following installed on your system:
* [Node.js](https://nodejs.org/) (which includes npm)

---

## Installation

1.  **Clone the repository** (if you haven't already):
    ```sh
    git clone [https://github.com/sparshagra/WhatIF_Chatbot-AI_powered_document_analysis.git](https://github.com/sparshagra/WhatIF_Chatbot-AI_powered_document_analysis.git)
    ```

2.  **Navigate to the frontend directory**:
    ```sh
    cd WhatIF_Chatbot-AI_powered_document_analysis/frontend
    ```

3.  **Install the required dependencies**:
    ```sh
    npm install
    ```
    *(If you use Yarn, you can use `yarn install`)*

---

## Available Scripts

In the project directory, you can run the following commands:

### `npm start`

Runs the app in development mode.
Open [http://localhost:3000](http://localhost:3000) (or the port specified in the console) to view it in your browser.

The page will reload automatically when you make code changes.

### `npm run build`

Builds the app for production to the `build` folder.
It correctly bundles the application in production mode and optimizes it for the best performance.

The build is minified, and the filenames include hashes for cache-busting.
